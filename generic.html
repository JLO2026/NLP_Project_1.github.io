Project

Here we collected articles from Event Registry API to collect articles on Tesla. Data below are results from using Vader, TextBlob, and BERT.

The purpose of this project was to test the ratings of reviews (positive, negative, or neutral) using some form of sentiment analysis. We did so using python libraries VADER, TextBlob, and BERT to do so. The image above compares the three results.

As you can see in the chart, the highest to lowest positive ratings goes from left to right, with most neutral being the opposite direction. What one could initially assume is that Textblob is more accurate as the reviews are more balanced. However, we can draw many assumptions from this. One more probable explanation is that TextBlob is more conservative in how they read what is positive, negative, or neutral. In our code, the polarity score for positive or negative is based on value 0, while VADER is using 0.05. That small difference defintely shows a change for our results. The explanation for BERT is that we were using a pre-trained model which was more prone to giving neutral results already, so we could not effect that. A easy fix would have been to use a model not pre-trained, but we did not have access.
